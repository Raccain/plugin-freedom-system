## Context to Preserve

**Key Decisions:**
- Plugin type: MIDI-triggered Sampler (hybrid instrument/effect)
- Core concept: Inspired by Eurorack Radio Music module - continuous folder playback with MIDI re-triggering
- Playback behavior: Always resumes from Start position (transport restart or MIDI trigger)
- Station changes: Queue new file, apply on next MIDI trigger (or immediately if free-running)
- Waveform display: Shows full file, darkens region before Start position
- File support: WAV, AIFF, MP3, FLAC, OGG - handle 100+ files
- Speed/Pitch: Coupled (tape-style), not time-stretching

**Technical Highlights:**
- Lazy loading for performance with large folders
- Stream large files from disk
- Click-free MIDI re-triggering with short fade-in
- Seamless looping
